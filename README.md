<h4 align="center">DoggyDog is a website created to help animals for adoption.</h4>

## Description
<table>
<tr>
<td>
 DoggyDog was born from the idea of helping animals for adoption. The purpose of this website is to collect information about the animals from different animals adoption centers. In this way, to give visibility to these animals so that people can get in touch with these centers.
</td>
</tr>
</table>
